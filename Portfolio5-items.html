Amazon Product API

Amazon advertising API

Overview

In this project, I built my personal API. This API is the easiest way to get access to product, price, sales rank and reviews data from Amazon. It was designed using React and Node.js (back-end server). There are two endpoints for this API: Get Amazon Search Results and Get Amazon Product Details.

The project is currently hosted by heroku, and can be found on Rapid API's website hub with the title: nnamdi-amazon-api. The API has three plans, The Basic plan (FREE), for only 500 requests a month. The Ultra-Plan charges $0.01 per 10,000 requests per month and the Next Ultra-Plan charges $0.10 per 50,000 requests per month.
