Amazon Product API

Amazon advertising API

Overview

I developed a personal API called the Amazon Product API as a web development project. This API provides users with access to product, price, sales rank, and reviews data from Amazon. It was built using React and Node.js, with two endpoints available: Get Amazon Search Results and Get Amazon Product Details.

The API is hosted by Heroku and can be found on Rapid API's website hub under the title "nnamdi-amazon-api." It is offered in three plans: a Basic plan that allows for 500 requests per month at no cost, an Ultra-Plan that charges $0.01 per 10,000 requests per month, and a Next Ultra-Plan that charges $0.10 per 50,000 requests per month.

This project demonstrates my skills in web development and API design, and the ability to create a functional and user-friendly product.
