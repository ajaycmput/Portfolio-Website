The title of my project

Stock API

Overview

Stock API is a stock forcasting program designed with python by me and my team (Goobers) for the Novermber 2021 Hackathon Contest. This program is intended to help make better decisions on whether to buy or sell the stocks the next day or so, based on the simple moving average models.

This project was Inspired by data forecasting algorithms and was aided by libraries from matplotlib and pandas along with data structure and some conditional statements.

This can perhaps be a building block in creating future forecasting models for stock.
